# Enterprise-Security-Operations-Lab
Enterprise Security Operations &amp; Vulnerability Management Lab using Active Directory, Nessus, Wazuh, and attack simulation
## Overview
This project simulates a real-world enterprise environment where I built, secured, and monitored an Active Directory infrastructure.

## Technologies Used
- Windows Server 2022
- Active Directory
- Hyper-V
- Nessus
- Wazuh
- Sysmon
- Kali Linux

## Goals
- Build enterprise infrastructure
- Perform vulnerability management
- Simulate real-world attacks
- Detect threats using SIEM tools
- Document and remediate security risks
## Day 1 - Domain Controller Setup
- Installed Windows Server 2022
- Configured static IP
- Installed AD DS role
- Promoted server to Domain Controller (lab.local)
